Ex11 - Description

Program that allocates and uses free store 

- new int gives pInt a new address to store integers
- new float gives pFloat a new address to store floats
- new double gives pDouble a new address to store doubles

- *pInt/*pFloat/*pDouble ~ populating the variables with number values on the freestore

- Display the pointer address and the value of that pointer on the Freestore.

- Release allocated memory by using delete
- Assign null values to pointers to avoid dangling pointers
