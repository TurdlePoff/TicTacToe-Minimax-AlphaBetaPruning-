Ex8 - Description

A program that detects arrow key press

- When arrow keys are pressed, 2 values are inputted simultaneously
	- The first value is 224 which is a non-showable 
	  character and represents the arrowkeys
		- 0xE0 = 224
		- ` = 96
	- The second value will be the value of the key that is pressed
		- 72 Up
		- 75 Left
		- 77 Right
		- 80 Down
