Ex5 - Description

- A program trying out the many str functions

- Fills buffers with different letters
- Terminates last item in buffer with null character

- strcopy - copies each letter from one buffer to another
- strncpy - copies characters from the specified point of 
	the array/pointed location to another buffer
- strcmp - compares 2 buffers and checks if they are bigger/less 
	than each other or equal to each other 
- strcat - adds on the letters from the buffer specified to another
- strcspn - finds a specific letter in the buffer
- strstr - searches for the start of the substring that is specified 
	and returns the address of the first letter
- strlen - counts the number of letters present in the buffer
